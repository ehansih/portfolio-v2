# Portfolio V2

Second portfolio site of **Harsh Vardhan Singh Chauhan** — Telecom Security specialist, 5G Architect, and Product Strategist with 16+ years of experience.

## Live Site
🌐 [saharsh1-projects.vercel.app](https://saharsh1-projects.vercel.app)

## About
Purple-accented modern portfolio inspired by [MoncyDev/Portfolio-Website](https://github.com/MoncyDev/Portfolio-Website). Built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools.

## Design
- **Background**: `#0b080c` deep dark
- **Accent**: `#c2a4ff` lavender purple
- **Font**: Geist (by Vercel)
- **FX**: Animated glowing blur circles, particle canvas, custom cursor, scroll-reveal

## Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — Particles, cursor, lightbox, scroll reveal
- **Vercel** — Deployment

## Features
- Animated pink/purple glowing circles (MoncyDev-style hero)
- Custom cursor with ring follower
- Hero with photo + split text layout
- Scroll-reveal animations
- Responsive mobile layout
- Domain expertise hover list
- Career timeline (16+ years)
- GitHub projects grid (9 repos)
- Photography lightbox gallery
- Contact form via Formspree

## Sections
- Hero Landing (photo centered, text on both sides)
- About + Stats
- Areas of Expertise
- Career Journey
- GitHub Projects
- Photography
- Contact

## Deployment

```bash
npm i -g vercel
vercel --prod
```
